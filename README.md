# Campus Life Super App 🎓🗺️

A multi-page web application designed to help students navigate campus life at North Park University. The app provides information about upcoming events, dining options, and an interactive campus map.

## Features

- Live weather data from Open-Meteo API 🌤️
- Upcoming campus events with expandable details
- Daily dining menu for Main Dining Hall and Cafeteria
- Interactive campus map with clickable location pins 📍

## ⚙️ Setup

### Prerequisites
- Web browser
- Live Server (VS Code extension) 

### Steps
1. **Clone the repository**
```bash
git clone https://github.com/annalena-project/campus-life-super-app.git
cd campus-life-super-app
```
2. **Open with Live Server**

Right-click on `index.html` in VS Code and select **Open with Live Server**

3. **Or visit the deployed site**

https://annalena-project.github.io/campus-life-super-app/
